# Бронь компьютера

Создать пустую папку на компьютере и клонировать в неё приложение используя команду : git clone git clone https://github.com/Ss1dan/ComputerBook.git
 
Перейти в корневую папку программы и запустить установку зависимостей: npm install
После установки запускаем сервер: npm start
 
Для проверки работоспособности откройте браузер или Postman и выполните Get-запрос: http://localhost:3000/api/computers/free
